# Project Name: SkillSphere -An Online Learning Platform

🔗 Live URL: https://skill-sphere-8ljw8411a-ajwadhussain38540-9801s-projects.vercel.app/
## Project Purpose

SkillSphere is a modern online learning platform where users can explore skill-based courses, view course details, and manage their profiles. The platform provides secure authentication using BetterAuth and Google Login while maintaining a clean, responsive, and user-friendly interface.
##  Key Features

-  Secure Authentication with BetterAuth
-  Google Sign In Support
-  Browse All Courses
-  Popular Courses Section
-  Trending Courses Section
-  Search Courses by Title
-  Protected Course Details Page
-  User Profile Page
-  Update Profile Information
-  Learning Tips Section
-  Top Instructors Section
-  Fully Responsive Design
-  Loading Spinner During Data Fetching
-  Toast Notifications
-  Custom 404 Not Found Page
